# MEVN-Boilerplate
A MEVN boilerplate with Tailwindcss and JWT authentication.

To get started with `npm`:
`cd backend && npm install` and
`cd frontend && npm install`

To get started with `yarn`:
`cd backend && yarn` and
`cd frontend && yarn`



**To start dev:**  

backend: `yarn dev`  

frontend: `yarn serve`

To build frontend: `yarn build`
